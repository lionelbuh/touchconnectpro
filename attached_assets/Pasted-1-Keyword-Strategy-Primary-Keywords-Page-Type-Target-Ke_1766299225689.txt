3️⃣ Blog Content Guidelines for AI Tool

Structure for AI-generated blog posts:

Title: ≤ 60 characters, include main keyword

Meta Description: ≤ 160 characters

H1: Repeat main keyword naturally

H2/H3: Include secondary/long-tail keywords

Content: Minimum 700–1200 words, informative, actionable

CTA: At least one CTA linking to mentor signup pages

Internal Linking: Each blog post should link to 2–3 relevant mentor profiles or category pages.

4️⃣ Technical SEO

Sitemap: /sitemap.xml

Robots.txt: Allow crawling of all pages

Structured Data: Implement JSON-LD schema for:

Mentor Profiles (Person + Organization)

Blog Articles (Article)

Reviews (Review)

Events (if mentorship sessions are bookable)

Example Structured Data for Mentor Profile (JSON-LD):

{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "Kaan Akkanat",
  "jobTitle": "Marketing Mentor for Startups",
  "worksFor": {
    "@type": "Organization",
    "name": "TouchConnectPro"
  },
  "url": "https://touchconnectpro.com/mentors/kaan-akkanat",
  "sameAs": ["LinkedIn URL if available"],
  "description": "Connect with Kaan Akkanat, an experienced marketing mentor helping startup founders grow their businesses."
}

5️⃣ Off-Page SEO Guidelines

Backlinks: Encourage mentors to link to their TouchConnectPro profile.

Guest Posts & PR: Publish startup guides on tech/startup blogs.

Social Media Sharing: Blog posts and mentor profiles should be shareable.

Community Engagement: Share insights in LinkedIn groups, startup forums, and communities.

6️⃣ Tracking & Analytics

Google Search Console: Track indexed pages & keyword performance

Google Analytics: Track traffic, user engagement, mentor page signups

Keyword Ranking: Use tools like Ahrefs, SEMrush, or free Rank Tracker
